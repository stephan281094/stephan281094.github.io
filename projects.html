• Scrollify.js

    24th of May, 2014

    Scrollify.js is a micro (0.263 kb) jQuery library that disables parent scrolling. Personally I hate it when I'm scrolling on a website inside some sort of container and when I reach the bottom of the scrolling container, the whole page continues with scrolling.

    That's why I decided to create a very tiny library that disables this. You can simply include a minified version of scrollify.js on your website, add a class to the container you want to apply scrollify to and call scrollify. Since it's such a small library that only took me half an hour to make, I decided to make this library open source. A lot of people cope with this problem and I think it'd be rude of me not to share something this simple and small.

    If you want to see the source code or if you want to improve the code, you're free to contribute to the Github repository here: Scrollify.js

    Technical details

    As the name 'scrollify.js' implies, it's a JavaScript library. However, I used coffeescript to build the library. You can see the code in the image above or find it at the git repository.

    You're probably wondering how the micro library works.
    Basically, when you hover over the element you're scrolling in, scrollify disables the parent's scroll. In most cases you want to disable scrolling on the HTML body element. But that's not always the case, which is why I've added the feature that enables you to set what element you want to disable for scrolling.

    You can do this by providing a parameter, like so: $('.scrollable').scrollify(".container");
